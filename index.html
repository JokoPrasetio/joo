Tentang

Halo! Saya Joko Prasetio, seorang Fullstack Web Developer asal Kota Lubuklinggau (Sumatera Selatan). Saya senang membangun aplikasi web dari awal — mulai dari perancangan antarmuka hingga deployment.

Saya menikmati proses menyederhanakan kebutuhan kompleks menjadi solusi digital yang elegan. Prinsip saya: “Koding itu bukan soal teknis saja, tapi bagaimana menyelesaikan masalah nyata dengan cara yang efisien dan bisa di-scale.”

Saat ini saya aktif mengembangkan sistem Hospital Information System, dan di waktu senggang sedang membangun chatbot berbasis AI. Saya juga terus belajar perkembangan teknologi baru seperti kecerdasan buatan.

Experience

Dengan pengalaman dalam berbagai proyek internal maupun freelance, saya terbiasa menggunakan Laravel sebagai Fullstack Developer—baik untuk pengembangan backend maupun frontend. Di sisi frontend, saya berpengalaman menggunakan Bootstrap, JavaScript, dan React. Saya juga memiliki keahlian dalam membangun sistem otentikasi, dashboard admin, REST API, serta integrasi sistem pembayaran dan notifikasi.

Selama Dua tahun terakhir, saya membangun sistem internal perusahaan, plugin undangan digital, dan dashboard analitik menggunakan PHP, Laravel, Bootstrap, dan JS modern.
